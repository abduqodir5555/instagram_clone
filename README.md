## O'rnatish
1. Repozitoriyani klonlash:
    ```bash
    git https://github.com/abduqodir5555/instagram_clone.git
    ```
2. Loyihani papkasiga o'tish:
    ```bash
    cd Instagram_project
    ```
3. Kerakli kutubxonalarni o'rnatish:
    ```bash
    pip install -r requirements.txt
    ```
## Hissa qo'shish
Hissa qo'shish uchun quyidagilarni bajaring:
1. Repozitoriyani fork qiling.
2. O'zingizning branchingizni yarating (`git checkout -b yangi-branch`).
3. O'zgartirishlaringizni commit qiling (`git commit -m 'O'zgartirishlar haqida qisqacha ma'lumot'`).
4. Branchingizni push qiling (`git push origin yangi-branch`).
5. Pull request yarating.
